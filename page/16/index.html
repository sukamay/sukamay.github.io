Tomcat配置

1. 给Mac安装jdk环境。

　　　 直接百度”jdk for Mac“，找到其中一个版本进行安装就可以了，安装完成后，打开终端，输入“java”或“javac”，如果出现java命令的使用说明，那么说明你的Mac上的jdk环境已经配置好，这个比起windows系统来说，简单方便很多。

2. 下载安装 Tomcat

​ 下载 Tomcat 地址（官方地址）：https://tomcat.apache.org/download-80.cgi

​ 下载文件：mac下载 zip 和 tar.gz 都行，我下载的是 tar.gz

3. 安装：

  1. 压缩之后直接拷贝到资源库中，压缩后的文件夹名称为Tomcat8，记录此位置:/Library/Tomcat8（方便后续环境配置）

  2. 打开终端，输入以下命令/Library/Tomcat8/bin/startup.sh，浏览器中输入localhost:8080，就可以看到以下效果:

    • 如果遇到提示No such file or directory错误提示，需要执行sudo chmod 755 Library/Tomcat/bin/*.sh，然后重新执行startup.sh，若出现permission denied 的情况，则在sudo模式下执行startup.sh，如果想停止Tomcat，直接执行shutdown.sh即可。在执行sudo时，必须使用带密码的用户。

​ 如果每次写路径比较麻烦，这个时候需要设置环境变量。

4. 设置 Tomcat 的环境变量:

ps：不要在root权限下设置

　　①使用vim编辑bash_profile，输入命令打开：vi .bash_profile(pico .bash_profile亦可)

　　②按 i进入编辑模式

​ ③输入Tomcat存储路径 export PATH="/Users/keso/Library/Tomcat8/bin:$PATH"（注意：此处路径是自己保存的 tomcat 存储路径）

　　④esc退出编辑模式，输入 :wq 保存并退出vim

​ ⑤更新配置的环境变量，输入命令：source .bash_profile

　　⑥终端中输入startup.sh， startup.sh后出现类似 “Permission denied” ，这个时候需要对目录进行权限设置：输入 sudo chmod 755 Library/Tomcat8/bin/*.sh 回车，设置文件的读写执行权限，参考第3步中的解决方案。

5. 设置完成

之后，输入 sudo startup.sh会显示启动, 打开 localhost:8080 可测试，出现上述页面。

关闭使用 shutdown.sh 即可;

到此，Tomcat 配置完成。

课前准备:

一:根据参与媒体人数，对应学生在互动课开始之前到达指定教室，并到达指定位置坐下等 待课程开始。
二:课程所需材料发放到课桌上摆放完毕
材料(需购置):

1.16k 素描纸:媒体及学生每人一张(我手中只有 8k 素描纸，需要裁剪成 16k，是否担心有 裁剪痕迹??)
2.水彩笔:媒体每人一盒(尽量购买不可水洗的彩笔)
3.2b 铅笔:媒体一人一只

4.橡皮:媒体一人一块 5.细两头黑色马克笔:媒体及学生一人一只(马克笔如图)

6.黑色签字笔:媒体与学生一人一只 三:黑板上板书:“美术互动课堂——新合完小 2016.11.17”为标题

正式流程:

1.开场(2min) 在座的朋友大家好，我是新合完小的项目老师，教授三年级和四年级的美术，非常高兴和大 家一起参与到这个美术的互动课程中来，希望在接下来的一个半小时，我们可以一起交流创 作出有趣的美术作品，同时，我会在大家的创作过程中随时给出指导，希望大家玩儿的开心。 2.媒体朋友与孩子的互动认识环节(2-3min)

现在呢，先请在场的媒体朋友一一走到孩子的身边(孩子此时已经一人一桌坐好了)，一对 一选择一位学生，跟她们进行 2 分钟的简单交流，互相认识一下~包括姓名、年级、年龄等。 3.安排互动课创作内容(10min) 师:今天啊，我们的美术互动课程是主题创作，题目是《我知道的你不知道的事》。在创作 之前，我们先来聊聊吧。首先我们来请一位学生代表说说自己的家在哪里? 生:新合。。。 师:那我们在座的同学看看你们身边，是不是有一位大朋友啊，他们来自哪里呢?不知道你 们刚才悄悄交流过没有，现在找几位朋友问问看?

媒体朋友:家乡地点和简介 师:那我们光知道地点，好像不够啊，我们来想想家乡有什么事情让我们印象深刻?比如说 一处风景、一种植物或者动物、一顿饭菜、一次与小朋友的玩耍等等、或者是一种色彩，让 你觉得可以代表家乡~我们请学生和我们的大朋友来分享一下，找找看待会儿你要画什么吧。 生:介绍家乡的特色
媒体朋友:介绍家乡特色 师:好，我们刚才已经回忆过让我们印象深刻的家乡景物。现在我们看看今天要怎么创作呢? 是用什么材料。看看你的桌上都有什么?
生:铅笔、橡皮、纸、彩笔。。 师:对，这就是我们今天要使用的材料，在纸上画出你知道的别人不知道的事，但注意要画 些你可以画出来的物品，或者是那个让你充满回忆，代表了家乡的东西。然后，我们看这张 纸是不是两面都是空白的，我们要在一面进行创作，另一面写上一些话，或者是你对家乡的 认识，或者是祝福的语言，然后，你和身边的大朋友就要进行作品的互赠啦~ 最后，在黑板上看一下创作步骤: 第一:确定创作内容(千万不要一想到画你自己知道的，新合的，家乡的样子就是一座山， 一棵树，一座房子，家乡可以是一种物品，如果画一种物品，一定注意构图饱满，小小一点 不好看啊，如果是一些色彩，可用回顾我们之前的马赛克，注意分割画面，填充代表家乡的 色彩等。)
1.充满色彩的画面(家乡给你的感觉，色彩分割时学的那些) 2.一片树叶、一朵花儿、一种吃的等等(注意饱满与装饰，或者用它做元素充满整个画面) 3.其他代表家乡的你能画出来的物品。 第二:铅笔起稿(下笔不要太重，不好修改，轻轻下笔，确定后可以再描一遍的，可以给画

面加外框~) 第三:彩铅填充色彩(色彩使用与搭配，用笔的轻与重会影响彩铅的显色程度哦) 第四:丰富画面效果(看看有什么需要补充的，可以把“我的家乡——一个美丽的地方”写 在画面上)
第五:在作品背面使用黑色签字笔，写上一些话。(祝福、介绍家乡) 4.创作环节(35min)
创作环节，老师巡堂，随时给出指导
