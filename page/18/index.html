  2. 安装npm

  3. 安装asar

  4. 进入staruml的包目录

  5. 解压app.asar

  6. 进入验证license的文件

  7. 修改源代码

    4. 输入 :wq 退回到原先的终端界面

  8. 重新打包替换原来的app.asar
